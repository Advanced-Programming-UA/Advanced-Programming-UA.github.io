Session 0 (07/10/2021)

This is the first session in a series of 10 labs covering various aspects of programming in C++; ranging from the most basic examples to more complicated topics introduced in the newer C++11, C++14, C++17 standards. Most things covered in the theoretical course notes will be applied in practice during these sessions. For starters you'll see an introduction to the more practical aspects of the labs: setting up the programming environment, version control, etc.

Session 1 (14/10/2021)

You will learn about the various types of variables such as built-in primitive types, structures and classes. Also covered are pointers and references to variables and the semantics of passing and retrieving data from functions as well as conversions between various types.

Read more »

Session 2 (21/10/2021)

This session covers construction and destruction of class objects and other related topics. You'll also see how operator overloading works and how it can be used for convenience. Additionally you will learn about function pointers, functors and function wrappers as well as C++11 lambda functions which all work together to facilitate some nifty constructions. Finally, we will have a look at the useful auto keyword.

Read more »

Session 3 (28/10/2021)

Today's lab session covers class inheritance and the use of abstract classes. Of special importance are the details concerning creation, destruction and copying of objects within class hierarchies. You'll apply these concepts in the context of two interesting and useful design patterns. Along the way you will learn a small & handy C++11 feature. The main exercise builds upon the function-plotter exercise you did last week.

Read more »

Session 4 (04/11/2021)

This session is all about exams!
You will first get an introduction to the project and learn how to integrate a GitHub repository with CircleCI.
Afterwards, you can make an exercise similar to the one you will get during the Tracing exam in January.

Session 5 (18/11/2021)

We will first start off with a small test to check your inheritance skills after last week's lesson. Afterwards, two subjects related to "safe programming" are the main topic of today's lab session: error handling through the use of exceptions and std features related to managing memory: smart pointers. We'll also cover a small addition to operator overloading: user-defined typecast operators.

Read more »

Session 6 (24/11/2020)

This week first covers the important concepts of move semantics and rvalue references. Following that, a variety of topics related to C++11 with accompanying reading work, are discussed. Also covered: two utility libraries for parsing commandline arguments and some interesting debugging tools to help improve your (project) C++ code.
