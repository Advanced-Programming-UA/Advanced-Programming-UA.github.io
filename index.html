Session 0
(06/10/2022)

This is the first session in a series of 10 labs covering various aspects of programming in C++; ranging from the most basic examples to more complicated topics. Most things covered in the theoretical course notes will be applied in practice during these sessions. For starters, you'll see an introduction to the more practical aspects of the labs: setting up the programming environment, version control, etc.

Session 1
(13/10/2022)

You will learn about the various types of variables such as built-in primitive types, structures and classes. Also covered are pointers and references to variables and the semantics of passing and retrieving data from functions as well as conversions between various types.

Read more »

Session 2
(20/10/2022)

This session covers construction and destruction of class objects and other related topics. You'll also see how operator overloading works and how it can be used for convenience. Additionally, you will learn about function pointers, functors and function wrappers as well as lambda functions, which all work together facilitate some nifty constructions. Finally, we will have a look at the useful auto keyword.

Read more »

Session 3
(03/11/2022)

Today's lab session covers class inheritance and the use of abstract classes. Of special importance are the details concerning creation, destruction and copying of objects within class hierarchies. Along the way you will learn a small & handy C++11 feature. The main exercise builds upon the function-plotter exercise you did last week.

Read more »

Session 4
(09/11/2022)

This session is all about exams!
You will first get an introduction to the project and afterwards, you will make an exercise similar to the one you will get during the Tracing exam in January.

Read more »

Session 5
(17/11/2022)

We will first start off with a small test to check your inheritance skills after session 3. Afterwards, two subjects related to "safe programming" are the main topic of today's lab session: error handling through the use of exceptions and std features related to managing memory: smart pointers. We'll also cover a small addition to operator overloading: user-defined typecast operators.

Read more »

Session 6
(24/11/2022)

This week first covers the important concepts of move semantics and rvalue references. Following that, a variety of topics that were introduced in C++11 with accompanying reading work, are discussed. Also covered: a utility library for parsing command-line arguments and some interesting debugging tools to help improve your (project) C++ code.

Read more »

Session 7
(01/12/2022)

This lab covers some of the more complex C++ features: pointers to member functions and more elaborate uses of C++11 lambda functions. We wrap up with a gentle introduction to C++ multithreading and two exercises.

Read more »

Session 8
(08/12/2022)

This lab session is entirely dedicated to the subject of C++ function templates. It's no exaggeration to say that templates are a very advanced topic. However, the introduction of templates to C++ in 1989 was a huge step towards the development of generic containers and algorithms in C++. A couple of examples and exercises should get you started.

Read more »

Session 9
(15/12/2021)

While the previous lab was entirely dedicated to the subject of C++ function templates, in this session we'll focus on class templates. While there are a lot of similarities to function templates, there are also important differences that have to be looked at. At the end, we'll also scratch the surface of template metaprogramming.
