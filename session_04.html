Session 4 (09/11/2022)

Summary

This session is all about exams!
You will first get an introduction to the project and afterwards, you will make an exercise similar to the one you will get during the Tracing exam in January.

Project

The project assignment itself and the corresponding presentation can be found here or on Blackboard.
During this session, you will have the opportunity to ask any questions you might have.

Tracing Exercise

The presentation containing the introduction to the tracing exercises can be found here or on Blackboard. The idea behind this session is for you to start making the tracing exercise that was used on the January 2022 exam (exam_2022_1), to properly get familiar with the concept behind the exercise and have an opportunity to ask questions. On the exam itself you will have 3 hours to complete this exercise, but this lab session is obviously shorter than that, and you still have to figure out how the exercise works. It's therefore not necessary to complete anything today or hand in your solution, since you can completely check it yourself using the object tracer. If you do want to get a better idea on how well you did, feel free to e-mail me containing your solution and I will correct it using the same checklist that was used during the actual exam. The repository that contains the Object Tracer can be found here, along with the generated Doxygen documentation here. In the presentation you can find all information necessary to complete the exercise and how to generate a correct solution.
